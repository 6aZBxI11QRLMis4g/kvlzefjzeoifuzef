Locatie / Omgeving

Voor een busreis naar Istrië werken wij samen met het gerenommeerde busbedrijf Interbus, en naar Dalmatië met Eurolines.  Bij onze busprijzen zijn de Royal Class-toeslag, bestemmingstoeslag, reserveringskosten en transfer inclusief. Daar het kan voorkomen dat één van deze toeslagen kan worden verhoogd behouden wij ons het recht om een prijsberekening door te voeren. U wordt hier altijd van op de hoogte gesteld.

Het is mogelijk om een losse busreis te boeken.
